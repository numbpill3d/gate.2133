1. OVERVIEW

Ghost of the BIOS is a retro-futuristic Neocities theme that simulates a haunted computer BIOS interface. The theme features dark monochrome colors with glowing green and purple accents, creating an atmosphere of digital paranoia and spectral computing.

Key Features

  • Authentic BIOS-style interface with boot sequence
  • Fully customizable color schemes via CSS variables
  • 5 complete pages: Home, Logs, Memories, Terminal, Corrupted Files
  • Interactive JavaScript components with sound effects
  • Responsive design that works on all screen sizes
  • Modular CSS architecture for easy customization
  • Accessibility features and reduced motion support
  • Ghost and glitch effects with performance optimization
COMPATIBILITY

This theme is designed for modern browsers and works best on Neocities. All effects degrade gracefully on older browsers.

2. INSTALLATION

Quick Setup

  1. Upload all files to your Neocities root directory
  2. Set index.html as your main page
  3. Customize colors in config.css (optional)
  4. Test all pages to ensure proper functionality

Required Files

index.html ← Main BIOS interface logs.html ← System logs viewer memories.html ← Memory/gallery page terminal.html ← Chat terminal corrupted.html ← File browser config.css ← Customization variables style.css ← Main stylesheet script.js ← Core JavaScript README.html ← This documentation
IMPORTANT

Do not rename the CSS or JS files as they are referenced by all HTML pages. If you must rename them, update all <link> and <script> tags accordingly.

3. FILE STRUCTURE

GHOST_BIOS_THEME/ ├── index.html # Main BIOS interface with system status ├── logs.html # System logs with real-time updates ├── memories.html # Memory viewer/gallery with tooltips ├── terminal.html # Interactive chat terminal ├── corrupted.html # File browser with corruption effects ├── config.css # Customization variables and themes ├── style.css # Main stylesheet with all components ├── script.js # Core JavaScript functionality └── README.html # Documentation (this file)

CSS Architecture

  • config.css - Contains all customizable CSS variables
  • style.css - Main styles using the config variables
  • Each page includes both files for consistent theming
  • Page-specific styles are included inline when needed

4. CUSTOMIZATION GUIDE

The theme uses CSS custom properties (variables) for easy customization. All customization is done in the config.css file.

4.1 Color Schemes

The default color scheme uses dark backgrounds with green and purple accents. You can easily change these by modifying the color variables:

Variable Default Value Description
--primary-bg #0a0a0a Main background color
--accent-green #00ff41 Primary accent (green glow)
--accent-purple #8a2be2 Secondary accent (ghost color)
--accent-red #ff0040 Error/warning color
--text-primary #c0c0c0 Main text color

Pre-built Color Themes

The config.css file includes several commented-out color themes. Uncomment one to use it:

AMBER TERMINAL
Classic amber monochrome
BLUE SCREEN
Windows BSOD style
MATRIX GREEN
Pure matrix aesthetic
CYBERPUNK PINK
Neon cyberpunk vibes
/* Example: Activate the Amber Terminal theme */ :root { --accent-green: #ff8000; --accent-purple: #ffaa00; --text-primary: #ffaa00; --text-bright: #ffdd00; }

4.2 Typography

Variable Default Value Description
--font-mono 'Courier New', Monaco, monospace Monospace font stack
--text-base 0.9rem Base font size
--text-lg 1.1rem Large text size

4.3 Visual Effects

You can enable or disable various visual effects:

Variable Values Description
--enable-glow block / none Enable/disable glow effects
--enable-glitch block / none Enable/disable glitch animations
--enable-scan-lines block / none Enable/disable CRT scanlines
--glow-intensity 10px Glow effect strength
--glitch-frequency 3s How often glitch occurs
/* Example: Disable all effects for better performance */ :root { --enable-glow: none; --enable-glitch: none; --enable-scan-lines: none; }

4.4 Responsive Settings

The theme includes responsive breakpoints and mobile optimizations:

--breakpoint-sm 480px Mobile breakpoint
--sidebar-width 250px Sidebar width on desktop

7. TROUBLESHOOTING

Common Issues

Boot Screen Stuck

Problem: Boot screen doesn't disappear
Solution: Press any key or wait 3 seconds. Check JavaScript console for errors.

Animations Not Working

Problem: Glitch effects or glows not visible
Solution: Check if effects are disabled in config.css or by reduced motion settings.

Mobile Display Issues

Problem: Layout broken on mobile
Solution: The theme automatically adjusts for mobile. Clear cache and reload.

Browser Compatibility

  • Chrome/Edge: Full compatibility with all effects
  • Firefox: Full compatibility with all effects
  • Safari: Most effects work, some WebAudio limitations
  • Mobile Browsers: Effects automatically reduced for performance

Debugging Tips

// Enable debug mode in browser console window.ghostBIOS.debugMode = true; // Check if CSS variables are loaded console.log(getComputedStyle(document.documentElement) .getPropertyValue('--accent-green')); // Test audio functionality window.ghostBIOS.playSystemSound();
